Elementor widgets for Team presentation and outline in awesome designs layouts. Presentation of your team members with predesigned widgets and customize as you wish with Elementor controls.

== Description ==
Presentation of the teams' members with Web Devise Elementor Team widget plugin. Use predesigned widgets to display members on your website quickly and awesomely. The plugin contains a bundle of controls that we all time upgrade making the process of the team displaying easy and quick. Customize a big part of things or use the Premium version to have advanced features of controls and more designs..

**The plugin includes different widgets with different designs**.

= Mobile optimized =
Team cards widgets optimized for mobile devices. Also, don’t forget to use integrated mobile options by Elementor to customize the mobile view as you wish.

= SEO friendly =
All code in widgets is done from an SEO point of view for good index by Google and perspective to easier integrate to any theme and according to your SEO strategy.

= Widgets =
Widgets included in this plugin you could easily control on the backend from the admin area of Elementor.

== Elementor Controls ==
Below is the list of controls that are included in the current version. Don’t forget to make updates to have up to date features:

- **Image and effects**. You could select a custom image for the output with control over effects, for example, Grayscale in Team1 widget.
- **Custom link**. Set up a custom link if you need to write down a custom URL address
- **Person Name**. Write down and customize a Team member name as you wish
- **Link text**. 
- **Typography**. Control to manage typography for the Team member Name
- **Color**. Elementor control for coloring texts
- **Social networks**. Some of the widgets are provided social links which you could set up for each team member
- **Content alignment**. Align content to make a nice look

The current version of the Product is workable with Elementor free and Elementor PRO
You could easily apply design styles to all Team members' widgets just by creating a CSS rule, it means if you need to change something in design in future you won’t need to customize each widget you just need to apply CSS rules to other Team members' widget. More detailed actions please find out in the FAQ.

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Pages > Add New
4. Press the 'Edit with Elementor' button.
5. Now you can drag and drop widgets (WebDevise section: Team1, 2, 3, ..) from the left panel onto the content area, as well as add new sections and columns that make up the page structure.
